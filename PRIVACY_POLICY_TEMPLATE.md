# 개인정보처리방침 - Weekly Gacha

**최종 업데이트: 2025년 12월 10일**

## 1. 개요

Weekly Gacha("앱", "서비스", "저희")는 사용자의 개인정보 보호를 매우 중요하게 생각합니다. 본 개인정보처리방침은 Weekly Gacha 앱을 사용할 때 수집, 사용 및 공개되는 정보에 대해 설명합니다.

## 2. 수집하는 정보

### 2.1 자동으로 수집되는 정보

**Google 계정 정보 (Google Sign-In 사용시):**
- 이메일 주소
- 사용자 이름
- 프로필 사진 URL
- Google 사용자 ID

**사용 데이터:**
- 앱 사용 통계
- 기기 정보 (기기 모델, 운영체제 버전)
- 앱 버전 정보

**광고 관련 정보 (Google AdMob):**
- 광고 ID
- 광고 노출 및 클릭 데이터
- 기기 정보 (광고 목적)

### 2.2 사용자가 제공하는 정보

**게임 진행 데이터:**
- 수집한 카드 정보
- 가챠 기록
- 사용한 쿠폰 정보
- 게임 내 설정

## 3. 정보 사용 목적

수집된 정보는 다음과 같은 목적으로 사용됩니다:

- **서비스 제공**: 앱 기능 제공 및 계정 관리
- **개인화**: 사용자 경험 맞춤화
- **클라우드 동기화**: 여러 기기 간 데이터 동기화
- **통계 분석**: 앱 개선 및 버그 수정
- **광고 표시**: Google AdMob을 통한 맞춤형 광고 제공
- **고객 지원**: 사용자 문의 및 기술 지원

## 4. 정보 공유 및 공개

저희는 다음과 같은 경우를 제외하고 사용자의 개인정보를 제3자와 공유하지 않습니다:

### 4.1 서비스 제공업체

**Firebase (Google LLC)**
- 용도: 사용자 인증, 데이터베이스, 클라우드 스토리지
- 개인정보처리방침: [Firebase Privacy Policy](https://firebase.google.com/support/privacy)

**Google AdMob (Google LLC)**
- 용도: 앱 내 광고 표시
- 개인정보처리방침: [AdMob Privacy Policy](https://support.google.com/admob/answer/6128543)

### 4.2 법적 요구사항

법률, 규정, 법적 절차 또는 정부 요청에 따라 정보를 공개해야 할 수 있습니다.

## 5. 데이터 보안

저희는 사용자 정보를 보호하기 위해 다음과 같은 보안 조치를 취합니다:

- **암호화**: 전송 중인 데이터는 SSL/TLS로 암호화
- **Firebase Security Rules**: 데이터베이스 접근 제어
- **Google 인증**: 안전한 OAuth 2.0 인증 사용

## 6. 데이터 보관 기간

- **계정 데이터**: 사용자가 계정을 삭제할 때까지 보관
- **게임 진행 데이터**: 계정 삭제 시 함께 삭제
- **사용 통계**: 익명화되어 최대 2년간 보관

## 7. 사용자 권리

사용자는 다음과 같은 권리를 가집니다:

### 7.1 접근 권한
수집된 개인정보에 대한 접근 요청

### 7.2 수정 권한
부정확한 개인정보의 수정 요청

### 7.3 삭제 권한
개인정보 삭제 요청 (계정 삭제)

### 7.4 데이터 이동 권한
수집된 데이터의 사본 요청

**권리 행사 방법**: 앱 내 설정 → 계정 관리 또는 이메일 문의

## 8. 어린이 개인정보 보호

Weekly Gacha는 모든 연령대를 대상으로 하며, 13세 미만 어린이의 개인정보를 의도적으로 수집하지 않습니다. 

부모님 또는 보호자께서는:
- 어린이의 앱 사용을 모니터링할 수 있습니다
- 13세 미만 어린이가 앱을 사용하는 경우 보호자의 동의가 필요합니다

## 9. 광고 및 제3자 서비스

### 9.1 Google AdMob
앱은 Google AdMob을 사용하여 광고를 표시합니다. AdMob은 다음을 수집할 수 있습니다:
- 광고 ID
- 기기 정보
- 위치 정보 (선택적)

### 9.2 광고 개인화 설정
사용자는 다음 방법으로 맞춤형 광고를 제어할 수 있습니다:
- **Android**: 설정 → Google → 광고 → 광고 맞춤설정 선택 해제
- 앱 내 설정에서 광고 개인화 옵션 조정

### 9.3 제3자 링크
앱에는 제3자 웹사이트 링크가 포함될 수 있습니다. 저희는 제3자 사이트의 개인정보 관행에 대해 책임지지 않습니다.

## 10. 쿠키 및 유사 기술

앱은 다음과 같은 기술을 사용합니다:
- **로컬 저장소**: 앱 설정 및 게임 진행 데이터 저장
- **세션 쿠키**: 사용자 세션 유지
- **분석 SDK**: 앱 사용 통계 수집

## 11. 국제 데이터 전송

사용자 데이터는 Firebase 서버에 저장되며, 이는 다음 위치에 있을 수 있습니다:
- 미국
- 유럽 연합
- 기타 Google Cloud Platform 지역

데이터 전송 시 적절한 보호 조치가 적용됩니다.

## 12. 개인정보처리방침 변경

본 개인정보처리방침은 필요에 따라 업데이트될 수 있습니다. 중요한 변경 사항이 있을 경우:
- 앱 내 알림
- 이메일 통지 (필요시)
- 최종 업데이트 날짜 표시

변경사항은 게시 즉시 효력이 발생합니다.

## 14. 법적 근거 (EU 사용자)

GDPR에 따라, 개인정보 처리의 법적 근거는 다음과 같습니다:
- **계약 이행**: 서비스 제공
- **정당한 이익**: 앱 개선 및 보안
- **동의**: 광고 개인화 (선택적)

## 15. 캘리포니아 주민 권리 (CCPA)

캘리포니아 주민은 다음 권리를 가집니다:
- 수집된 개인정보에 대한 정보 요청
- 개인정보 삭제 요청
- 개인정보 판매 거부 (해당 없음 - 저희는 개인정보를 판매하지 않습니다)

## 16. 데이터 처리 요약

| 데이터 유형 | 수집 목적 | 보관 기간 | 제3자 공유 |
|------------|----------|----------|-----------|
| Google 계정 정보 | 인증, 계정 관리 | 계정 삭제시까지 | Firebase |
| 게임 진행 데이터 | 서비스 제공 | 계정 삭제시까지 | Firebase |
| 광고 ID | 광고 표시 | AdMob 정책 따름 | Google AdMob |
| 사용 통계 | 앱 개선 | 최대 2년 (익명화) | Firebase Analytics |

## 17. 동의 철회

사용자는 언제든지 다음 방법으로 동의를 철회할 수 있습니다:
1. **앱 내 설정** → 계정 관리 → 계정 삭제
2. **이메일 요청**: 개발자에게 직접 연락

계정 삭제 시 모든 개인정보가 30일 이내에 삭제됩니다.
